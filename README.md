## Run Locally

Clone the project

```
  git clone https://github.com/Priya2994/Restoguru.git
```

Go to the project directory

```
  cd Restoguru
```

Install dependencies

```
  npm install
```

Start the server

```
  npm run dev
```

Compiles and minifies for production
```
npm run build
```
